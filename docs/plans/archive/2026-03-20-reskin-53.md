# #53 Reskin — 7 Colour/Style Directions Implementation Plan

> **For Claude:** REQUIRED SUB-SKILL: Use superpowers:executing-plans to implement this plan task-by-task.

**Goal:** Create 7 colour/typography reskins of homepage-53-muevete-inspired.html (files 71–77), keeping all content, layout, and sections identical — only changing palette, fonts, and texture.

**Architecture:** Each reskin is a mechanical copy-then-replace operation. Copy #53, run a series of `sed` find-and-replace commands for colours + fonts, swap the Google Fonts `<link>`, then inject direction-specific texture CSS. All 7 can be built in parallel since they share no dependencies. Each agent reads #53 once, applies its replacement map, adds its texture, and writes the output file.

**Tech Stack:** Bash (`cp`, `sed -i ''` on macOS), HTML/CSS (inline style edits for texture additions)

**Step 3: Replace fonts**

